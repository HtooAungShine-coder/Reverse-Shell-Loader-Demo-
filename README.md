# Reverse-Shell-Loader-Demo
This is a demonetization of how a no Metasploit used Reverse Shell Loader would look like in real world.

What this code contain : 
Windows Networking with C++
Spawns "cmd.exe" hidden (SW_HIDE) with stream redirection enabled
Holds execution open until cmd.exe exits

What this code does not contain :
Obfuscation
Encryption / Decryption
Anti-Virus Evasion

Or any other things that will " perfectly " act as a malware.
My grandma's Windows Xp would flag this code in a second So
Read and Learn From this as Educational Purposes Only.
